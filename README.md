## 楽天のサマーインターンで作成したアプリ

### 解決したい課題
コロナ渦で人々の交流が減り、オフラインでの交流がなくなった。オンラインでイベント等に参加することはできるが、一人で参加することが多くなった。
### 解決策
オンラインで手軽に他社と交流して熱狂できる音声プラットフォームアプリを開発しよう

とういう流れで開発しました
